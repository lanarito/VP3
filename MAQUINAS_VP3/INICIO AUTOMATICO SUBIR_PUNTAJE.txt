============================================================
   VP3 - INICIO AUTOMATICO CON WATCHDOG
============================================================

NUEVO SISTEMA A PRUEBA DE FALLOS:
Si subir_puntajes.exe se cierra por cualquier motivo, el WATCHDOG
lo reinicia automaticamente. Asi NUNCA quedan records sin sincronizar.


CONFIGURACION (solo se hace una vez por maquina):

1. Presiona Windows + R
2. Escribe: shell:startup
3. Presiona Enter (se abre la carpeta de inicio de Windows)
4. Crea un acceso directo a:
   WATCHDOG_invisible.vbs

   (este archivo esta en la carpeta MAQUINAS_VP3)

5. Pega el acceso directo en la carpeta de inicio

IMPORTANTE: Si tenian un acceso directo viejo de subir_puntajes.exe
en la carpeta de inicio, BORRARLO y reemplazarlo por el de WATCHDOG_invisible.vbs


QUE HACE EL WATCHDOG:
- Inicia subir_puntajes.exe en silencio (segundo plano)
- Si el .exe se cierra por crash, error o lo que sea, lo reinicia
- Espera 5 segundos y lo arranca de nuevo
- Mantiene el sistema SIEMPRE sincronizando

ARCHIVOS DE LOG (para verificar que funciona):
- vp3_heartbeat.txt        ← Ultima vez que el script estuvo vivo
- vp3_script_log.txt       ← Eventos del script (inicios, sincronizaciones, errores)
- watchdog_log.txt         ← Reinicios del watchdog

Si el script esta sano, vp3_heartbeat.txt se actualiza cada 5 minutos.
Si pasaron mas de 10 minutos sin actualizar, algo paso.


SISTEMA DE SEGURIDAD INTEGRADO:
Ademas del watchdog, el script tambien hace una sincronizacion forzada
cada 10 minutos como red de seguridad, por si NVRAM cambia sin actualizar
su fecha de modificacion.


============================================================
RESUMEN: Solo prendes la maquina, y todo funciona solo.
Si algo se rompe, se arregla solo.
============================================================
